# Rust HTTP Server 🦀

This is a personal project designed to dive deep into **Rust** fundamentals and low-level systems programming. The main goal is to build a functional HTTP server from scratch, putting memory safety and concurrency concepts into practice.

## 🚀 Project Goal
I am using this repository to showcase my technical skills in Rust, applying knowledge gained through my engineering studies and self-taught exploration. The server focuses on:
* Efficient TCP connection management.
* Processing HTTP requests and responses.
* Implementing a Thread Pool to handle multiple concurrent requests.

## 🛠️ Technologies
* **Language:** Rust
* **Tools:** Cargo
* **Dev Environment:** Arch Linux 🐧

## 📋 Learning Roadmap
- [x] Initial project setup and TCP stream handling.
- [x] Basic HTTP protocol implementation (GET requests).
- [ ] Error handling and server robustness.
- [x] Refactoring for concurrency using a Thread Pool.
- [ ] Support for static files and other methods (POST, PUT).
